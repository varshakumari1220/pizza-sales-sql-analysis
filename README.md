# pizza-sales-sql-analysis
SQL-based Pizza Sales Analysis using MySQL to derive key business insights. Includes queries from basic to advanced for revenue, top-selling pizzas, order trends, and customer preferences. Uses JOIN, GROUP BY, and aggregate functions to analyze real-world sales data efficiently.

🍕 Pizza Sales Analysis using SQL

#Project Overview
This project analyzes pizza sales data using SQL to extract meaningful business insights such as revenue trends, customer preferences, and sales performance.

# Objectives
- Calculate total revenue and total orders
- Identify top-selling pizzas and categories
- Analyze order patterns (hour-wise, date-wise)
- Understand customer preferences (size, type)

#Tools & Technologies
- SQL (MySQL)
- Database Management System

#Key Insights
- Most popular pizza types and sizes
- Peak order hours
- Category-wise sales distribution
- Revenue contribution analysis
